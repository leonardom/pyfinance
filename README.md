# pyfinance

## Create virtual environment
```shell
python -m venv .venv
```

## Install dependencies
```shell
pip install -r requirements.py
````

## Run stocks
```shell
python stocks.py <TICKER-1> <TICKER-2> ... <TICKER-n>
```

## Run fiis
```shell
python fiis.py <TICKER-1> <TICKER-2> ... <TICKER-n>
```
